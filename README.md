## Fizz Buzz
**Fizz Buzz Code** in _C#_

```for (int i = 1; i<= 100; i++)
{
    if (i % 3 == 0 && i % 5 == 0)
    {
      Console.WriteLine("Fizz Buzz");
    }
    else if (i % 3 == 0)
    {
      Console.Writeline("Fizz");
    }
    else if (i % 5 == 0)
    {
      Console.WriteLine("Buzz");
    }
    else
    {
      Console.WriteLine(i);
    }
}
```
